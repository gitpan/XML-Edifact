XML::Edifact - an approach towards XML/EDI as a prototype in perl

release 0.40 - stabilisation of edicooked syntax

Michael Koehne, ( kraehe@copyleft.de )

v0.40 release

XML::Edifact is a set of perl scripts, hopefully becoming a module, for translating EDIFACT into XML. This 0.40 version open the portability track. Starting with a minor bugfix ( thanks to Werner F.C. Bruns ) and questions for a W32 port on a DIN meeting in Frankfurt.
